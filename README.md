# AutoGen Lesson 5 — Financial Analysis

A portfolio project built from Lesson 5 of the DeepLearning.AI course **AI Agentic Design Patterns with AutoGen**.

This app demonstrates a multi-agent financial analysis workflow using AutoGen and Streamlit.

## Features

- Multi-agent collaboration with AutoGen
- Financial stock analysis
- Choose 2 stocks from predefined options
- AI-generated chart
- Agent conversation visibility
- Streamlit UI

## Tech Stack

- Python
- AutoGen
- Streamlit
- OpenAI API
- yFinance
- Matplotlib

## Available Stocks

- NVDA
- TSLA
- AAPL
- MSFT
- AMZN
- GOOGL
